# 🚀 웹 배포 가이드

## 완성되면 이렇게 됩니다!

✅ 누구나 접속 가능한 URL (예: https://osh-law-chatbot.streamlit.app)
✅ 예쁜 웹 UI
✅ 실시간 법령 검색 + AI 답변

---

## 📋 준비물

1. GitHub 계정 (없으면 https://github.com 에서 가입)
2. 아래 2개 파일:
   - `app.py` (메인 코드)
   - `requirements.txt` (라이브러리 목록)

---

## 🔧 Step 1: GitHub 저장소 만들기

### 1-1. GitHub 로그인
👉 https://github.com 접속 후 로그인

### 1-2. 새 저장소(Repository) 만들기
1. 오른쪽 상단 **"+"** 버튼 클릭
2. **"New repository"** 선택
3. 설정:
   - Repository name: `osh-law-chatbot` (원하는 이름)
   - Public 선택 (무료 배포하려면 Public이어야 함)
   - "Add a README file" 체크
4. **"Create repository"** 클릭

### 1-3. 파일 업로드
1. 저장소 페이지에서 **"Add file"** → **"Upload files"** 클릭
2. `app.py`와 `requirements.txt` 파일을 드래그앤드롭
3. 아래 **"Commit changes"** 클릭

---

## ☁️ Step 2: Streamlit Cloud 배포

### 2-1. Streamlit Cloud 가입
👉 https://share.streamlit.io 접속
- **"Sign up with GitHub"** 클릭 (GitHub으로 가입)

### 2-2. 앱 배포하기
1. 로그인 후 **"New app"** 클릭
2. 설정:
   - Repository: `본인아이디/osh-law-chatbot` 선택
   - Branch: `main`
   - Main file path: `app.py`
3. **"Deploy!"** 클릭

### 2-3. 배포 완료!
- 약 5~10분 후 배포 완료
- URL 생성: `https://본인아이디-osh-law-chatbot-app-xxxxx.streamlit.app`

---

## ⚙️ Step 3: (선택) API 키 환경변수 설정

보안을 위해 API 키를 환경변수로 설정할 수 있어요.

1. Streamlit Cloud 앱 설정 페이지
2. **"Settings"** → **"Secrets"**
3. 아래 내용 추가:
```toml
ANTHROPIC_API_KEY = "sk-ant-xxxxx"
```

그러면 사용자가 API 키 입력 안 해도 됨!
(이 기능을 쓰려면 app.py 코드 수정 필요)

---

## 🎉 완료!

이제 URL을 고객에게 공유하면 끝!

---

## ❓ 문제 해결

### "ModuleNotFoundError" 에러
→ requirements.txt에 해당 라이브러리 추가

### 배포가 안 됨
→ GitHub 저장소가 Public인지 확인

### 로딩이 너무 느림
→ 처음 접속 시 모델 로딩 때문에 느림 (정상)
→ 이후 접속은 빨라짐
